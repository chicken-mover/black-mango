:mod:`blackmango.ui.keyboard` --- Key bindings and binding lookups
==================================================================

.. automodule:: blackmango.ui.keyboard

.. autofunction:: check

Data values
-----------

.. data:: bindings

    A dictionary that maps action names to a list of acceptable key presses for
    that action. Because these values are subject to change, it is advides that
    you consult the module source code to see what the current list of possible
    keys are.
